## Practice Resources

This repository contains the code examples, prompts, and supporting assets used throughout the *AI Side Hustle* book.

All chapter-specific resources are organized in `chapter-resources.md`. If you would like to follow along with the exercises, copy prompts, or use the sample code from the book, start there.

The file includes:

* Chapter-by-chapter prompts
* Example code
* Configuration settings
* Reference links and additional resources

New materials and updates will be added as the book evolves, so this repository serves as the most up-to-date source for practical resources.
